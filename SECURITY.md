# Security Policy

## Supported Versions

S.I.L.K. is an active, rolling-release project. We only support the latest version of the codebase.

| Version | Supported |
| :--- | :--- |
| **Main Branch** (Latest) | :white_check_mark: |
| Old Commits | :x: |

## Reporting a Vulnerability

If you discover a security vulnerability within S.I.L.K. (e.g., a way to bypass command permissions, leak data, or crash the bot), **please do NOT open a public issue.**

We appreciate your help in keeping this project safe. Please report sensitive bugs via the following methods:

1.  **Email:** shifosan.official@gmail.com
2.  **Discord:** DM the creator **ShifoSan (shifosan)** directly.
3.  **Private Issue:** If you are a collaborator, open a private advisory.

### What to include in your report:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Any relevant screenshots or logs.

We will acknowledge your report within 48 hours and strive to patch the vulnerability as soon as possible.
